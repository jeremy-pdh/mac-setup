# mac-setup

My macOS development environment setup guide.

## Package Manager
- **Homebrew**: Core package manager for installing applications and dependencies

## Shell Configuration
- **zsh**: Default shell with custom configuration
- **.zshrc**: Shell initialization file with aliases, functions, and environment variables

## Applications
- Development tools and IDEs
- Productivity applications
- Utilities and system tools

## Configuration Files
- Dotfiles for environment customization
- Shell profiles and rc files
- Application preferences

## Installation
1. Install Homebrew
2. Clone this repository
3. Run setup scripts
4. Configure shell and applications

## Contents
- `Brewfile`: Homebrew packages and applications
- `.zshrc`: Zsh configuration
- Configuration scripts and dotfiles
